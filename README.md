# Franky Cars instalation and running

## App


Clone this repo and enter frontend folder

```bash
cd frontend
```

Use npm install

```bash
npm i
```
After the instalation, start app using

```bash
npm start
```

## Server


Clone this repo and enter backend folder

```bash
cd backend
```

Use npm install

```bash
npm i
```
After the instalation, start server using

```bash
npm start
```
